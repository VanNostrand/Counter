Counter

Magic the Gathering Life Counter for PC (Windows, Linux, Mac)

Download this project as a .zip file Download this project as a tar.gz file

Welcome to the Counter project site.

Counter is a simple life counter program for Magic the Gathering, licensed under the GNU GPLv3, which makes the software free as in beer and speech.

It has the possibility to save the current life value into a .txt file as requested by the folks at manabase.de. This feature is used to read the counter into a video streaming software.

If you need a counter for more than one player or if you need additional counters like poison counters just open another instance of the program, contribute a patch or motivate me.

Counter uses a cross-platform framework, thus works on Windows, Mac and Linux. There are only Windows builds available at the time though, but compiling the code on a Unix system is not complicated. There are three versions:

  • v1.0 - The standard version (Download).
  • v1.1a - The standard version with coloured LCD (Download).
  • v1.1a manabase-edition - A special, compact version for manabase.de available in the branch "manabase-edition" (Download).

For further details see the README.md file.
